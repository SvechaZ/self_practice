Problem :
check if n is a prime number and prime number is positive.
print YES if n is a prime number. Otherwise, print NO.
the followings are not prime numbers:

"string", "2", NaN, Infinity, -3, null, ...

A prime number is a positive integer (> 1) 
that is divisible only by 1 and itself.

ในโจทย์ต้องการให้หาจำนวนเฉพาะ และเป็นค่าบวก หนูเลยเริ่มจากการเช็คค่าก่อนว่าค่าที่ input มา เป็น Number และเป็นค่าคืนที่ไม่ใช่จำนวนเต็มหรือทศนิยมหรือไม่
โดยการใช้ typeof เพื่อกันกรณีที่ input เป็นอย่างอื่น เช่น string , NaN และเมื่อเช็คเสร็จแล้ว เลยทำเงื่อไขต้อไป คือ ต้องการแค่บวก เลยใส่ x <= 1 เพื่อบอกให้ใส่ค่าที่มากกว่า 1 หรือเท่ากับ 1 ตามนิยาม 
ถ้าใส่น้อยกว่า 1 ก็จะขึ้นข้อความระบุให้ใส่เลขที่มากกว่า 1 และจึงไปทำเงื่อไขที่เช็คว่าเป็น prime number หรือไม่ คือ การใช้ for loop โดยหนูประกาศ IsPrime เป็น true ก่อน เพื่อเก็บสถานะ หรือไปแก้ค่า IsPrime
ในส่วนของ for loop ต้องกำหนด i = 2 เพราะ จำนวนเแพาะค่าบวก เริ่มที่ 2 และ i ต้องน้อยกว่า x เพราะเป็นหลักการของจำนวนเฉพาะที่ต้องได้ด้วยตัวมันเองและ 1
ในเงื่อนไข x % i === 0 เพราะ ค่า x ที่ใส่เข้ามาต้องหารลงตัวกับ i (จะไม่มีเศษ)  และที่จำเป็นต้องใส่ IsPrime = flase เพราะต้องเช็ก loop เพื่อไม่ให้หลังจบ loop พิม true (ไม่งั้นอาจจะ run แต่ true)
จากนั้นก็ใช้ break ในการหยุด loop เมื่อเจอไม่ใช่ prime number และถ้า x เป็น prime number ก็จะออกจาก for และ console มาเป็น YES
